# postit

## これなに？
ドメインモデルやクラス図などを表示するアプリ。  
PLANTUMLのようなもの。  

## PLANTUMLとの違い
- ブラウザだけあれば図を作成できる(javaとかgraphvizとか不要)
- サーバサイドが無いので情報流出とかしない(PLANTUMLのWebAPIとの違い)
- 出力がSVGなのでブログとかに貼りやすい(PLANTUMLでもで可能ですが...)
